================================================================================
  ENDPOINTS PRINCIPAIS
================================================================================

CRIAR ALUNO:
  POST /api/admin/alunos
  Body: { nome, email, whatsapp, senha }
  Requer: Token + Professor

LISTAR ALUNOS:
  GET /api/admin/alunos?busca=nome
  Requer: Token + Professor

BLOQUEAR ALUNO:
  PUT /api/admin/alunos/:id/status
  Body: { ativo: false }
  Requer: Token + Professor

REMOVER ALUNO (SOFT DELETE):
  DELETE /api/admin/alunos/:id
  Requer: Token + Professor

EDITAR ACESSO ÀS TRILHAS:
  PUT /api/admin/alunos/:id/trilhas
  Body: { trilhas_ids: [1, 2, 3] }
  Requer: Token + Professor

DELETAR TRILHA/MÓDULO/LIÇÃO:
  DELETE /api/trilhas|modulos|licoes/:id
  Requer: Token + Professor

================================================================================
  TROUBLESHOOTING RÁPIDO
================================================================================

"Email já cadastrado"
  → Use outro email ou delete aluno anterior

"Acesso negado"
  → Faça login como professor

"Token inválido"
  → Obtenha novo token via /api/auth/login

"Aluno não aparece"
  → Verifique se ativo = true no banco

"Erro ao criar"
  → Verifique validation errors nos logs

Dados não restauram?
  → Use: UPDATE users SET ativo = TRUE WHERE id = ...
